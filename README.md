## 🎯 Project Goal

The goal of this project is to create a **modern, clean, and high-converting laundromat website** that:

* Provides a smooth and intuitive user experience
* Clearly explains services, pricing, and process
* Builds trust through testimonials and strong visuals
* Encourages users to visit the location or contact the business
* Works seamlessly across all devices (mobile-first design)

---

## 🛠️ Tech Stack

* **Next.js (App Router)** – Framework for building fast, scalable React applications
* **Tailwind CSS** – Utility-first styling for rapid UI development
* **Framer Motion** – Smooth animations and transitions
* **Lucide React** – Modern icon library
* **Swiper.js** – Carousel/slider for testimonials

---

## 🚀 Features

* ⚡ Built with **Next.js (App Router)**
* 🎨 Clean **modern UI (Glass + Soft Gradients)**
* 📱 Fully **responsive design**
* 🎬 Smooth animations using **Framer Motion**
* 🎠 Interactive **testimonials carousel (Swiper.js)**
* 📍 Integrated **map & contact sections**
* 📞 Click-to-call functionality
* 🧩 Modular & reusable components

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/bubbles-laundromat.git
cd bubbles-laundromat
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
