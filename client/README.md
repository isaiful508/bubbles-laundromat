# Bubbles Laundromat Website

A modern, premium Next.js 15 website for Bubbles Laundromat in Oklahoma City.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 🏗️ Production Build

```bash
npm run build
npm start
```

## 📦 Tech Stack

- **Next.js 15** – App Router with TypeScript
- **Tailwind CSS 3** – Utility-first styling
- **Framer Motion 12** – Animations & transitions
- **Lucide React** – Icons
- **DM Serif Display + Outfit** – Typography

## 🔒 Security Features

- HTTP Security Headers (CSP, X-Frame-Options, X-XSS-Protection)
- Input sanitization (XSS prevention)
- Client-side rate limiting on contact form
- Email & name validation
- No third-party analytics or ad trackers

## 🎨 Customization

- Edit colors in `tailwind.config.ts`
- Update business info in `app/layout.tsx` (SEO metadata)
- Replace video source in `components/VideoSection.tsx`
- Add real email endpoint in `components/Contact.tsx`
